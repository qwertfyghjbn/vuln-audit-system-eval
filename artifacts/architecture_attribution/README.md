# Architecture Attribution Artifacts

这个目录用于保存 `Architecture Attribution Experiment` 的原始运行证据。

后续每个运行批次应至少记录：

- `run_manifest.json`
- `case_family_matrix.tsv`
- `task_input/` 或输入物化说明
- 原始系统输出
- 规范化中间结果
- 成本、耗时、tool call、上下文规模记录

本目录属于证据层，不作为主要人读入口。人读规划和总结应写入 `reports/systems/architecture-attribution/`。
